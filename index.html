* Yapp

Yapp is a full stack clone of Yelp. Yapp allows for a user to search for and review restaurants with full CRUD functionality. Yapp was developed using Ruby on Rails, PostgreSQL, JavaScript, React, and Redux.

Go to Yapp ReadMe

* Trip it

Trip it is a MERN stack application; I developed this app with three other engineers, focusing on the frontend. Trip it allows for a user to organize trips with other users by creating trips and trip events with full CRUD functionality. Trip it uses MongoDB, Mongoose, Express, JavaScript, React and Redux.

Go to Trip It ReadMe

PinPoint Perfection (In development)

PinPoint Perfection is a single page application built solely from JavaScript. PinPoint Perfection allows a user to select an oil pattern, input user specs, and add their bowling balls to get the best suggestion on which bowling ball the user should start with on the selected oil pattern.
